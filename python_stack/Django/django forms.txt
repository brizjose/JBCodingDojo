Django with Forms

1.
The property is request.POST if the method that hits the route is a post, and request.GET if the method that hits the route is a get.

2.
The structure is:

<form action="/blogs/create" method="post">
	{% csrf_token %}
	Blog Name: <input type="text" name="name" placeholder="blog name">
    Description: <textarea name="desc" placeholder="description"></textarea>
	<input type="submit" value="submit">
</form>

3.
This is needed to secure the data; goes in the html within the <form></form>

	{% csrf_token %}

4.
Add the url to /app/urls.py

url(r'^create$', views.create)

5.
Add the method in /app/views.py

from django.shortcuts import render, HttpResponse, redirect
def create(request):
	if request.method == "POST":
		print "*"*50
		print request.POST
		print request.POST['name']
		print request.POST['desc']
		request.session['name'] = "test"  # more on session below
		print "*"*50
		return redirect("/")
	else:
		return redirect("/")

6. Key terms so far:
	
	request.POST    --- data from POST request
	request.GET	--- data from GET request
	request.method  --- returns the method/HTTP verb associated with the request
	{% csrf_token %} --- prevents cross-site request forgery, place it on the html within the <form></form> capturing the data
