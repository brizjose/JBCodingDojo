bcrypt

used for password hashing

step1.

create hash:

>>> import bcrypt
>>> hash1 = bcrypt.hashpw('test'.encode(), bcrypt.gensalt())
>>> print hash1
$2b$12$Wdc2qwiP6u0WdQdKwmer7.DMIcY6q76GxvrJgaodnpRDmpP8mwkDa


step2.

compare password with hash:

>>> bcrypt.checkpw('test'.encode(), hash1.encode())
True
