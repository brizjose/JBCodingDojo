1. setting up a virtual environment

run the following code to install a virtual environment:

-------------------------------------------------
| Mac/Linux: | pip install virtualenv           |
-------------+-----------------------------------
| Windows:   | python -m pip install virtualenv |
-------------------------------------------------

2. navigate to your environments directory

$ cd ~/Documents/CodingDojo/python_stack/myEnvironments

3. create a new virtual environment

In this example, we named our environment with the python version (2), and the major dependency (Flask)

-------------------------------------------------
| Mac/Linux: | virtualenv flaskEnv           |
-------------+-----------------------------------
| Windows:   | python -m virtualenv flaskEnv |
-------------------------------------------------

4. activating the environment

While in the parent directory to our virtual environments (myEnvironments), activate the virtual environment by typing:

------------------------------------------------------------------
| Mac/Linux: | source flaskEnv/bin/activate                   |
-------------+----------------------------------------------------
| Windows command prompt :   | call flaskEnv/Scripts/activate |
| Windows 10 command prompt :   | . flaskEnv/Scripts/activate |
------------------------------------------------------------------
| Windows git bash :  |  source flaskEnv/Scripts/activate     |
------------------------------------------------------------------

5. once in the environment:

The (flaskEnv) indicates that your virtualenv is now active. If we install any dependencies they will install into our local flaskEnv folder:

(flaskEnv) $ pip install flask

6. we get out of the myEnvironments folder

cd ..

7. we navigate into the folder where our project sits

cd OjeksO\Ejksl

8.  we work!

9. deactivate

go back to normal environment;

$ deactivate
